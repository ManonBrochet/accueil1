# Frontend JSP

Bienvenue sur la page d'accueil du frontend JSP.

## À propos

Application frontend pour la plateforme JSP créée par Manon.

## Fonctionnement de la page d'accueil

La page d'accueil est le point d'entrée principal de l'application. Elle offre une interface conviviale permettant aux utilisateurs de :

- Consulter les informations générales sur la plateforme
- Accéder aux fonctionnalités principales de l'application
- Se connecter ou créer un compte
- Naviguer vers les différentes sections (cours, quiz, événements)

La page est construite avec React et utilise Vite comme outil de build pour des performances optimales.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run dev
```

## Build

```bash
npm run build
```
